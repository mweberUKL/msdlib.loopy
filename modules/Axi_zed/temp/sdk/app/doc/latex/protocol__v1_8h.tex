\subsection{Detailed Description}
Generic protocol encoder and decoder. Delegates calls to the respective protocol decoder for the protocol version the message was sent with. The protocol version is always stored in the same field of the header. Calls to encoder procedures are delegated to an encoder of the protocol version stored in the constants file (which has been specified by the user).
